Math.abs					абсолютное значение (модуль)
Math.ceil(value)			возвращает наименьшее целое число с плавающей точкой, которое не меньше value
Math.floor(value)			возвращает наибольшее целое число, которое не больше value
Math.max					возвращает максимальное число
Math.min					возвращает минимальное число
Math.pow(a, b): 			возвращает число a, возведенное в степень b
Math.random					возвращает случайное число от 0.0 до 1.0				
Math.rint(value)			возвращает число, которое представляет ближайшее к числу value целое число  
Math.round()				возвращает число, округленное до ближайшего целого числа
